## Summary

- 讨论并实现了 AI Project OS 的一键卸载能力。
- 新增 `ai-project-os uninstall`，并支持通过 `install.sh` 转发远程卸载命令。
- 将版本推进到 `v0.3.0`。

## Decisions

- `uninstall` 默认只删除已归属且未修改的 Project OS 文件。
- 修改过或未归属的同名文件默认保留，避免误删用户原有内容。
- `--force` 用于显式删除修改过或未归属的固定路径文件。

## Notes

- 卸载前会备份删除的文件到 `.project-os-backups/uninstall-*`。
- 目录只会在为空时删除，不递归删除用户额外内容。
- symlink 路径会被拒绝，避免跟随链接删除或备份到项目外。
